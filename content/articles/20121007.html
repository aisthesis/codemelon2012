Playground: Clouds

10-07-2012

This playground will explore how to create 3D clouds using CSS3 and JavaScript. The experiment is based on Jaume Sánchez's CSS3D Clouds.

As you move your mouse, the clouds will rotate. Scrolling can be used to zoom in or out. And pressing the spacebar will generate a new cloud configuration.

The clouds are generated by placing a .png of a cloud onto 3d backdrops positioned using random number generators.

I changed the speed variable of the original tutorial so that the clouds don't always rotate in the same direction. Now that I have it set up here, I'll try to find other ways in which to expand on this great idea.
