Class Overview

Java implementation of the linear programming algorithm provided in Cormen, Leiserson, Rivest and Stein, Introduction to Algorithms, 3rd ed., ch. 29. In this implementation, the objective function and inequality constraints must be specified when a Simplex object is created (i.e., as arguments to its constructor). The getResult() method will then generate and return an array of type double that provides an optimal solution, or it will return null if the solution is unbounded. If there are multiple solutions to the linear program specified when a Simplex object is created, getResult() will return one such solution.
