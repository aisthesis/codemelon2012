WeightedGraph

Known Indirect Subclasses

Class Overview

Interface specifying the required methods for weighted graphs.

In addition to real-valued edge weights, the edges of a weighted graph may contain various integer-valued edge properties. The user can specify in the graph constructor how many such properties are needed (if any). Weighted graphs also provide support for a real-valued vertex weight.

public abstract int edgeProperties ()

Determine the number of edge properties stored in the graph. Edge properties are used to store integer values associated with an edge. For example, it may be useful to associate a color property with an edge and store in the edge the integer corresponding to the given color.

Returns
  • an integer corresponding to the number of edge properties.

public abstract boolean insert (Edge e, double weight)

Insert an edge into the graph while also specifying its weight. Returns true iff the edge is not already present in the graph. Does nothing and returns false if the graph already contains the given edge.

e Edge to be inserted

Returns
  • a boolean value showing whether or not the operation was successful. True is returned if the edge could be inserted, and false is returned if the insertion failed.
