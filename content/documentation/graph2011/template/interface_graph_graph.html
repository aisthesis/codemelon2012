Public Methods

public abstract int edges ()

Determine the number of edges in the graph

Returns
  • an integer corresponding to the number of edges in the graph.

public abstract int[] getAdjacencies (int from)

Determine the adjacencies of a given vertex.

Parameters
from Integer value of the vertex for which adjacencies are to be determined. A vertex u is considered adjacent to from only if from dominates u.
Returns
  • an array of integers representing all vertices u s.t. (from, u) is an edge in the calling graph.

public abstract Edge[] getEdges ()

Retrieve all edges in the graph.

Returns
  • an array of Edge objects representing all edges in the calling graph.

public abstract Edge[] getEdges (int from)

Retrieve all edges dominated by a given vertex.

Parameters
from Integer value of the dominating vertex. The method returns all edges of which this vertex is the head or, in another terminology, dominated by the given vertex.
Returns
  • an array of Edge objects representing all edges of the form (from, u) where from is the input argument and u may be any vertex in the graph.

public abstract boolean hasEdge (Edge e)

Determine whether or not the graph contains a given edge.

Parameters
e Edge object whose presence in the graph is to be evaluated. The method will return true iff the graph contains an Edge e1 such that e.equals(e1).
Returns
    true if the graph contains the given edge and false if it does not.

public abstract boolean hasEdge (int from, int to)

Determine whether or not the graph contains a given edge.

Parameters
from Integer specifying the head (dominating) vertex for the edge whose presence is to be determined.
to Integer specifying the tail (dominated) vertex for the edge whose presence is to be determined.
Returns
    true if the graph contains the edge (from, to) and false if it does not.

public abstract boolean insert (Edge e)

Add an edge to the calling graph. Returns true and inserts the edge if the edge is not already present. If the given edge is present, this method returns false and does not modify the graph.

In an undirected graph, insert(e1) will have the same effect as insert(e2) if the head vertex of e1 is the tail of e2 and the tail vertex of e1 is the head of e2.

Parameters
e Edge object to be inserted.
Returns
    true iff the graph does not contain the given edge and the insertion is successful.

public abstract boolean insert (int from, int to)

Add an edge to the calling graph. Returns true and inserts the edge if the edge is not already present. If the given edge is present, this method returns false and does not modify the graph.

In an undirected graph, insert(from, to) has the same effect as insert(to, from).

Parameters
from Integer corresponding to the head of the edge to be inserted or, in another terminology, the vertex dominating the edge to be inserted.
to Integer corresponding to the tail of the edge to be inserted or, in another terminology, the vertex dominated by the edge to be inserted.
Returns
    true iff the graph does not contain the edge (from, to) and the insertion is successful.

public abstract boolean remove (Edge e)

Remove an edge. Returns true and removes the edge if the edge is found in the graph. If the given edge is not found, this method returns false and does not modify the graph.

In an undirected graph, remove(e1) will have the same effect as remove(e2) if the head vertex of e1 is the tail of e2 and the tail vertex of e1 is the head of e2.

Parameters
e Edge object to be removed.
Returns
    true iff the graph contains the given edge when the method is called and its removal is successful.

public abstract boolean remove (int from, int to)

Remove an edge. Returns true and removes the edge if it is found. If the given edge is not found, the method returns false and does not modify the graph.

In an undirected graph, remove(from, to) has the same effect as insert(to, from).

Parameters
from Integer corresponding to the head of the edge to be removed or, in another terminology, the vertex dominating the edge to be removed.
to Integer corresponding to the tail of the edge to be removed or, in another terminology, the vertex dominated by the edge to be removed.
Returns
    true iff the graph contains the edge (from, to) when the method is called and its removal is successful.

public abstract Graph transpose ()

Get the transpose of a graph.

Returns
  • a Graph object that is the transpose of the calling graph.

Determine the number of vertices in the graph

Returns
  • an integer corresponding to the number of vertices in the graph.
