Class Overview

A DisjointSet maintains a dynamic collection of disjoint sets, each of which is identified by a canonical member. A datastructure of this kind is needed in order to implement Kruskal's algorithm for finding a minimum spanning tree in a weighted graph. This version of the disjoint set datastructure includes only the basic operations needed for the implementation of graph algorithms and follows the simple union technique of CLRS 21.2 (pp. 564ff.) rather than the faster but more complicated techniques of 21.4. For optimal performance of Kruskal's algorithm, the current implementation needs to be improved.

The constructor creates a specified number (n in the following example) of sets, each having a single integer member: {0}, {1}, ..., {n - 1}. Additional methods then allow us to create unions of such sets and, following any sequence of union operations, to determine the set to which a given integer in the appropriate range belongs.

Public Constructors

public DisjointSet (int size)

Construct size disjoint sets, each containing a single integer between 0 and size - 1. The representative element returned by FindSet(int) is initially (prior to any union operations) simply the sole element of the set.

Throws
  • java.lang.IllegalArgumentException if the size argument is negative.

Public Methods

public int findSet (int member)

Identify the (unique) set containing member. The integers i and j are members of the same set within a DisjointSet object ds iff the equality ds.findSet(i) == ds.findSet(j) holds.

Parameters
member integer whose containing set is to be identified.
Returns
  • an integer corresponding to the canonical representative of the containing set.

public boolean union (int i, int j)

Merges the sets of which i and j are members. For any disjoint set ds and integers i and j in the appropriate range, execution of the statement ds.union(i, j) guarantees that subsequently the equality ds.findSet(i) == ds.findSet(j) will hold.

If i and j are already contained in the same set, this method will return false and do nothing.

Parameters
i an element determining (by virtue of being any member) one of the 2 sets to be merged.
j an element determining (by virtue of being any member) the other of the 2 sets to be merged.
Returns
  • a boolean value of true if i and j were members of distinct sets prior to execution of the union and false if i and j were already contained in the same set.
