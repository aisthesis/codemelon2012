Class Overview

Weighted edges are edges with support for a weight of type double. Like edges, weighted edges are immutable.

Public Constructors

public WeightedEdge (int from, int to, boolean charRep, double weight)

Construct a weighted edge from vertex from to vertex to with weight weight using character representation of vertices if charRep is set to true or integer representation if charRep is false. The inherited charRep() method of a weighted edge constructed in this way will return the value of charRep.

If charRep is set to true, client programs will likely prefer to use characters in the specification of the head and tail vertices. For example: WeightedEdge e = new WeightedEdge('r', 'u', true, 3.14159);

Throws
  • java.lang.IllegalArgumentException if either of the following conditions holds:
    1. 1) the from or to argument is negative.
    2. 2) charRep is true and both from and to are neither in the range 'a' - 'z' nor 'A' - 'Z'. This includes the case where one vertex is in the range 'a' - 'z' and the other is in the range 'A' - 'Z'.

public WeightedEdge (int from, int to, double weight)

Construct a weighted edge from vertex from to vertex to with weight weight using integer representation of vertices. The inherited charRep() method of a weighted edge constructed in this way will return false.

Throws
  • java.lang.IllegalArgumentException if the from or to argument is negative.

public int compareTo (WeightedEdge e)

Compare the weights of 2 weighted edges. Returns 0 if the edges have equal weights, 1 if the weight of the calling edge is greater than the weight of edge e, or -1 if the weight of the calling edge is less than the weight of edge e.

public String toString ()

Overrides the toString() method of graph.Edge. Returns a string of the form "(u, v w)" representing the calling edge and its weight w. If the edge represents vertices as characters (as determined by its inherited charRep() method), u and v will be the corresponding letters. Otherwise, u and v will be the corresponding integers.
